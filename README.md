# 엑셀 업체 위치 지도 - 내장 데이터 HTML 버전

이 버전은 엑셀 데이터를 `index.html` 안에 내장했기 때문에 웹페이지에서 엑셀 파일을 업로드하지 않아도 바로 지도가 표시됩니다.

## 포함 데이터

- 시트: `2026년 동반위`
- 표시 업체 수: 28개
- 표시 항목: 회사명, 주소, 업종, 위도, 경도
- 제외 항목: 담당자명, 연락처

## GitHub Pages 적용 방법

1. 이 ZIP의 압축을 풉니다.
2. GitHub 저장소 `map`으로 이동합니다.
3. 기존 `index.html`을 이 파일로 교체합니다.
   - `Add file` → `Upload files`로 올리면 같은 파일명이라 덮어쓰기됩니다.
4. `Commit changes`를 누릅니다.
5. 1~5분 후 아래 주소로 접속합니다.

```text
https://binah3740.github.io/map/
```

## 기존 출장거리 지도와 함께 쓰기

현재 HTML 안에 다음 링크가 포함되어 있습니다.

```text
출장거리-2_2026년 동반위_kakao_map.html
```

저장소에 해당 파일이 그대로 있으면 왼쪽 메뉴의 `출장거리 지도 열기` 버튼으로 이동할 수 있습니다.
